# ChatDevelopers App

It's an app realtime with Websockets using tech as Node, Express, and Socket.io with vanilla JS

## Install dependencies

```c
npm i
```

Now you can open it in: [http://localhost:3000](http://localhost:3000)

## Run in localhost

```
npm run dev
```

## Run in production

```
npm start
```
